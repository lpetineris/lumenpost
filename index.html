Perguntas frequentes

Sim. Os posts e o perfil ficam salvos na nuvem via Supabase, vinculados ao seu ID de usuário. Nenhum outro usuário tem acesso. O texto do tema e a imagem são enviados à IA apenas para gerar o rascunho.

Sim. O Lumen Post funciona em computador, tablet e celular. Os dados ficam salvos na nuvem e sincronizam automaticamente.

Sim. Quando você faz upload de uma imagem, ela é enviada para a IA junto com o tema. A IA analisa visualmente e cria um post baseado no que vê, tornando o conteúdo muito mais específico para aquela foto.

Sim. Selecione mais de uma rede antes de salvar — o sistema cria um post independente para cada rede. Após salvar, as redes voltam a ficar desmarcadas. O mesmo conteúdo pode ser salvo na mesma rede em datas diferentes.

No Banco de Posts, clique em "✉ Aprovar por email" no card desejado, insira o email do aprovador e clique em Enviar. O post muda para "Aguardando aprovação" e o aprovador recebe o texto por email. É possível enviar para um novo aprovador a qualquer momento.

Se você definir uma data ao salvar, o post vai automaticamente para aquela data no Calendário. Também é possível adicionar manualmente clicando no "+" em qualquer dia. O mesmo conteúdo pode ser agendado em datas diferentes.

No Banco de Posts, clique em "📅 Alterar data". Também é possível clicar no post no Calendário e usar o botão "📅 Alterar data" no modal. O post é movido automaticamente para a nova data.

Cada post é independente. Excluir pelo Banco remove aquele registro e sua entrada no Calendário naquela data. Excluir pelo Calendário remove apenas daquela data específica, sem afetar outras datas onde o mesmo conteúdo esteja agendado.

Clique em Regenerar — a IA cria uma nova versão. Se ainda não satisfizer, descreva o tema com mais detalhes, mude o tom ou troque a imagem. O botão Limpar zera o gerador para começar do zero.

Na versão atual, o Lumen Post suporta um perfil por vez. Para trocar de empresa, basta atualizar a aba Perfil e salvar.

Durante o período beta, o acesso é livre sem limite. Após o lançamento haverá planos com limites mensais.
